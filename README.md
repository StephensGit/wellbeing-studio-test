# wellbeing-studio-test

## Project setup, below is instructions to get the application running locally on your own computer:

### Use git clone to clone the applications repository

* cd [repository-directory]

* yarn install

* yarn serve

### The application will start by in your default browser of choice after these instructions

The project uses Vue and bootstrap 4 to create a single page application. It also makes use of Vue router.
