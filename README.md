# Wellbeing Studio

This application was designed and developed for fun to get experience ussing Vue.js


Required technologies:
- Bootstrap 4
- Vue.js
Create new project using Vue-CLI ( https://cli.vuejs.org/guide/ ).
Build one page using Bootstrap4 css framework ( bonus points for
https://bootstrap-vue.org/docs but plain Bootstrap 4 will do as well ) to match attached design.
Page has to be responsive.
Push project to VCS of your choice.
Bonus points for:
- Add Vue router and build one more page linked in the top nav menu
